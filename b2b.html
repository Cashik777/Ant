Часті питання B2B

Який мінімальний обсяг замовлення?

Мінімальне замовлення — 2 кг на місяць. Цього достатньо для невеликої кав'ярні чи офісу.

Чи можна замовити зразки для дегустації?

Так! Надсилаємо до 3 зразків по 100г безкоштовно. Це допоможе обрати ідеальний сорт для вашого закладу.

Як працює навчання бариста?

Для партнерів рівня "Бізнес" і вище — безкоштовний 3-годинний майстер-клас: налаштування обладнання, техніка приготування, латте-арт.

Які документи ви надаєте?

Повний пакет для ФОП та ТОВ: рахунок-фактура, видаткова накладна, акт виконаних робіт, сертифікати на продукцію.
